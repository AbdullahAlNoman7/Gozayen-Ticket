### Web Automation Gozayen-Ticket For Single Person

### Project Scenerio
1) Visit GoZayen
2) Click one way
3) Enter From destination
4) Enter to destination
5) Select date
6) Select travel class
7) Hit the serach button
### Technology Used
- Tools: Selenium
- Language: Python
- Editor: PhCharm 
### How to use this project
- Clone this project 
- pip install -r requirements.txt
